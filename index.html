Un navigateur bloque les appels cross-origin sauf si TheHive renvoie les bons en-têtes. Votre administrateur TheHive doit autoriser l'origine de cette app via play.http.cors / les en-têtes du reverse-proxy (Access-Control-Allow-Origin, -Headers: authorization,content-type). Si vous ne pouvez pas l'activer, faites tourner l'app derrière un proxy qui ajoute la clé API côté serveur.
